# SkillForge

> Forjando habilidades para una IA más útil.

SkillForge es una colección abierta de skills para IA creadas y compartidas por Lorspi. Cada skill está diseñada para resolver problemas reales, automatizar tareas o ampliar las capacidades de agentes inteligentes.

El objetivo de este repositorio es ofrecer recursos prácticos, reutilizables y fáciles de adaptar para que desarrolladores, creadores y equipos puedan construir mejores experiencias con inteligencia artificial.

## ¿Qué es una skill?

Una skill es un conjunto de instrucciones, conocimiento especializado, contexto y buenas prácticas que permite a un agente de IA ejecutar una tarea específica de manera más efectiva.

Las skills de este repositorio están diseñadas para ser reutilizables, fáciles de integrar y basadas en experiencias reales de desarrollo.

| Skill | Descripción |

| [wompi-payments](./skills/wompi-payments) | Skill para integrar Wompi (pasarela de pagos colombiana) en proyectos con Convex backend. Cubre configuración, widget de checkout, firma de integridad, webhook y procesamiento de pagos. |

## Filosofía

Las skills publicadas en SkillForge siguen algunos principios simples:

- 🧩 **Reutilizables**: diseñadas para ser utilizadas en múltiples proyectos.
- 🎯 **Enfocadas**: cada skill resuelve un problema específico.
- 🚀 **Prácticas**: construidas a partir de casos de uso reales.
- 🤝 **Abiertas**: compartidas para beneficio de toda la comunidad.
- 🧠 **Compatibles con IA moderna**: pensadas para agentes, asistentes y herramientas impulsadas por LLMs.
